# AI-ассистент "Tehna" — Frontend 🤖💼

Frontend - часть интеллектуальной HR-платформы, разработанной в рамках Международного хакатона "Технострелка 2026" (Нижний Новгород) для кейса "1221 Systems".

## 📋 Обзор проекта
Основная цель платформы - автоматизировать процессы управления персоналом, предоставляя сотрудникам мгновенные и достоверные ответы на корпоративные вопросы (например, об отпускных, заработной плате, политике компании) на основе загруженных документов (PDF, DOCX) и записей в базе данных.

** Наша команда заняла 4-е место на хакатоне!** 🏆

## ⚙️ Основные функции (на стороне интерфейса)
* ** Интеллектуальный интерфейс чата: ** Чистый, интуитивно понятный пользовательский интерфейс, созданный для беспрепятственного общения с AI-талисманом.
* ** Ответы с учетом контекста:** Отображает точные данные о сотрудниках (например, конкретные оставшиеся дни отпуска), а не просто общий текст.
* ** Указание источника:** Показывает, из какого именно абзаца документа ИИ извлек ответ.
* ** Адаптивный дизайн:** Оптимизирован как для веб-просмотра, так и для просмотра с мобильных устройств.
